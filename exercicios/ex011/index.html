Imagens dinâmicas

Tente abrir esse site em diferentes dispositivos ou simplesmente aumente e diminua o tamanho do seu navegado para que veja o funcionamento da imagem dinâmica!

Para criarmos imagens dinâmicas basta adicionarmos a tag <img> com a nossa imagem principal dentro de uma tag <picture>, após isso basta adicionarmos as outras imagens em ordem decrescente logo acima com os respectivos atributos:
<source media="(max-width: ?)" srcset="?" type="image/?">.
E tudo isso dentro da tag <picture>.

Reproduzindo áudio

Vamos aprender a reproduzir áudios com HTML5.

Para colocarmos áudios dentro de nossa página HTML utilizamos a tag <audio> seguindo de alguns parâmetros:
<audio preload="metadata" autoplay controls loop>

  • preload=
    • metadata(Vai carregar apenas as informações do arquivo Ex: Tamanho, tempo, informações de direito e etc.)
    • none(Não carregará absolutamente nada até que o usuario clique no play.)
    • auto(padrão)(Vai carregar o arquivo inteiro mesmo que o usuário não aperte no play.)
  • controls(vai apresentar o player na tela. Necessário que esse atributo seja colocado dentro da tag <áudio>.
  • autoplay(Vai iniciar a reprodução do áudio assim que a página for carregada.)
  • loop(Vai fazer com que o áudio se repita eternamente sempre que chegar ao fim.)

Dentro da tag <audio>, adicionamos vários <source> com formatos diferentes do mesmo áudio. Coloque na parte de cima o seu formato favorito. Os demais só serão carregados caso o de cima falhe. caso todos falhem, criamos um paragrafo que permite o dowload do arquivo MP3 para ouvir no player padrão do dispositivo.
