Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (Não semântica)

Nesta frase temos um termo em negrito utilizando a tag <strong> (Semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico utilizando a tag <i> (Não semântica)

Nesta frase temos um termo em ênfase utilizando a tag <em> (Semântica)

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

Texto grande e pequeno

Nesse parágrafo também estamos criando um texto grande e um texto pequano utilizando as tags <big> e <small>

Contudo a tag <big> ja é obsoleta pois não é semântica e não faz sentido um texto grande , já a tag <small> é semântica.

Texto deletado

Podemos criar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Para isso usamos a tag <del> que é semântica.

Texto inserido

Podemos também marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois usando a tag <ins> (Semântica)

Existe também o sublinhado usando a tag <u> mas essa tag não é semântica.

Texto sobrescrito

Com o texto sobrescrito podemos expressar coisa como uma elevação em uma equação: x20+3 ou x30-10 (Numeros pequenos em cima)

Para isso utilizamos a tag <sup> (Semântica)

Texto subscrito

E com o texto subscrito podemos expressar coisas como uma fórmula química: H2O (Numeros pequenos em baixo)

Para isso utilizamos a tag <sub> ( Semântica)
