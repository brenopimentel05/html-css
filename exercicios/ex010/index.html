Usando links externos

Links externos

Você pode acessar o meu repositório público no Github

Você também pode acessar o link do nosso canal no YouTube

Para criarmos um link/âncora utilizamos a tag <a> onde no parâmetro "href" devemos aplicar a URL do site desejado completa e usarmos os parâmetros "target= "_blank" e "rel= "external"

para que possamos criar links externos (links que não se sobrepõem sobre esta página, a página principal, pois são outros links sem ligação com o site principal)

Links internos

Siga agora para a segunda página

Siga também para a terceira página

Para que poossamos criar links internos basta aplicarmos a tag <a> da mesma forma que os links externos mas dessa vez no parâmetro "target" usamos o valor "_self" e no parâmetro "rel" usamos o valor "next" ou "prev"

Para criarmos páginas internas podemos criar outros arquivos .html dentro da própria pasta do documento principal ou criar uma nova pasta e dentro dela adicionar o novo arquivo .html, mas dessa forma é necessário que no parâmero "href" coloquemos o caminho certo utilizando CTRL + SPACE
Para alternarmos entre links internos caso ele esteja na mesma pasta que o documento principal basta usarmos os parâmetros "target" e "rel" corretamente mas caso o link esteja em pastas diferentes é necessário usarmos no "href" " ./ " caso o arquivo do link esteva na mesma pasta ou " ../ " caso o arquivo esteja fora da mesma pasta

Links para dowload

Para criarmos um link para dowload basta adicionarmos o endereço do dowload no parâmetro "href" e adicionarmos os parâmetros "dowload" com o endereço do dowload e o parâmetro "type" para adicionarmos o valor do tipo de arquivo.
